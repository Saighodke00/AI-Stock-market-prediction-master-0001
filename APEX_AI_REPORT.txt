SUBJECT: PROJECT WORKING & ARCHITECTURE OVERVIEW

--------------------------------------------------------------------------------
1. EXECUTIVE SUMMARY
--------------------------------------------------------------------------------
APEX AI is a high-frequency predictive platform designed for both Swing and 
Intraday trading. It utilizes Deep Learning (Temporal Fusion Transformers) 
to forecast price trajectories and validates them through a multi-stage 
confluence "Gate" system.

--------------------------------------------------------------------------------
2. TECHNICAL WORKING (THE PROCESS)
--------------------------------------------------------------------------------
The "Working" of the platform follows a strict 4-stage pipeline:

STAGE A: DATA INGESTION & NOISE REDUCTION
- Sources: Live OHLCV from yfinance + Macro Indices (VIX, S&P500, NSEI).
- Denoising: Uses Wavelet Transforms to remove market volatility "noise" 
  before the AI sees it.
- Fraction Differentiation: Ensures data is stationary while preserving the 
  "long-range memory" of price trends.

STAGE B: INFERENCE ENGINE (THE BRAIN)
- Model: Pre-trained TFT/Keras/TFLite models.
- Output: Generates a probabilistic "Confidence Cone" rather than a single 
  price point.
- Targets: Provides P10 (Bear), P50 (Base), and P90 (Bull) price projections.

STAGE C: CONFLUENCE GATING (THE SAFETY)
- No signal is emitted unless it passes all 4 "Gates":
  1. Attention Gate (Weighting recent patterns).
  2. Cone Gate (Statistically valid movements).
  3. Sentiment Gate (Live news headline support).
  4. Technical Gate (RSI/MACD alignment).

STAGE D: TERMINAL RENDERING (THE UI)
- Integration: TradingView-style Candlestick & Volume charts.
- Interaction: Dynamic crosshair sync with real-time OHLCV legend.
- Execution: Position sizing via Kelly Criterion based on signal confidence.

--------------------------------------------------------------------------------
3. RECENT IMPLEMENTATION (THE CANDLESTICK UPGRADE)
--------------------------------------------------------------------------------
The platform has recently been upgraded from basic line charts to a professional 
TradingView Terminal experience:
- Library: lightweight-charts (High-performance canvas rendering).
- Feature: Forecast Trajectories are now drawn as colored dashed lines 
  directly on the price grid.
- Alignment: Fixed timeframe handling (Unix timestamps for Intraday vs 
  ISO-dates for Swing).

--------------------------------------------------------------------------------
4. ARCHITECTURE OVERVIEW
--------------------------------------------------------------------------------
- FRONTEND: React + TypeScript + TailwindCSS.
- BACKEND: FastAPI + Uvicorn (Inference Server).
- CACHING: Redis (Timeframe-aware caching v6).
- MODELS: Keras (.keras) and TensorFlow Lite (.tflite).
